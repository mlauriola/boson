An ODF Common Code is any code that is used in two or more systems or has some significant high use. In the IT Games context, this applies to Games systems and ODF users. As these systems generate and distribute all the results related information during Gamestime, any system that is a receiver of such information should be aligned in terms of Common Codes.

The default character set is UTF-8. All characters in all supported languages are used, including accented characters.

All Games systems using any or all ODF Common Codes should support the format as defined in Olympic Data Feed Common Codes Definition document.

RSC Codes are totally different and are defined separately. Each RSC ODF common code is defined uniquely throughout all the common codes tables. That is, each RSC code is only defined to the table/level it belongs (i.e Discipline RSCs are defined only in the Discipline table).
